## init vue project

```shell
vue init vue-router-practice
cd vue-router-practice
npm install
```

## 安裝 vue-router

用 npm 或 yarn 來安裝 vue-router

```shell
npm i -S vue-router
```

```shell
yarn add vue-router
```

### run

```shell
npm run dev
```
